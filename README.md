# Unity Slot Machine Game

## Overview

A casino-style Slot Machine game built in Unity using UI-based reels, ScriptableObjects, coroutines, and object-oriented programming principles.

## Features

* 3-Reel Slot Machine
* Betting System (10 / 50 / 100)
* Currency Management
* Sequential Reel Stopping
* Lever Animation
* Reel Spin Sound Effects
* Win Sound Effects
* Win Popup Animation
* ScriptableObject-Based Symbol Data
* Modular Architecture

## Tech Stack

* Unity
* C#
* TextMeshPro
* ScriptableObjects
* Coroutines

## Project Structure

Scripts are organized into:

* Core
* Reels
* UI
* Audio
* Data

## Gameplay Rules

1. Select a bet amount.
2. Press Spin.
3. Reels spin and stop sequentially.
4. Matching symbols on the center payline award a payout.
5. Payout = Bet × Symbol Multiplier.
